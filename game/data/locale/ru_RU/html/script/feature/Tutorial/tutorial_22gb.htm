[Перевод Класса Эльфийского Волшебника]:
Для того чтобы стать Спокровителем Стихий, Эльфийский Волшебник должен успешно завершить три задания. Испытание Ученого доступно только игрокам с уровнем 35 и выше. Начало этого задания связано с встречей с Магистром Мириен в Дион Касл Тауне. Свидетельство Жизни может быть получено только игроком с уровнем 37 и выше. Начало этого задания связано с встречей с Мастером Кардиеном в Дион Касл Тауне. Наконец, Испытание Призывателя доступно только игроку с уровнем 39 и выше. Начало этого задания связано с встречей с Верховным Призывателем Галатеей в Деревне Глудин.
