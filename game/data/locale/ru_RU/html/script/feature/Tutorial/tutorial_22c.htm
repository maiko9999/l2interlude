[Перевод Класса Человеческого Волшебника]:
Волшебник уровня 40 может выполнить переход на второй класс и стать Чародеем/Чародейкой, Некромантом или Чернокнижником. Для выполнения второго классового перехода игрок должен успешно завершить три задания. Первое из трех заданий может быть выполнено только персонажем уровня 35 и выше. Если вы хотите увидеть более подробную информацию, пожалуйста, выберите интересующий вас класс.
