[Трансфер класса Человек-Рыцарь]:
Рыцарь, достигший 40-го уровня, может приступить ко вторичному трансферу, чтобы стать Паладином или Темным Авенджером. Для выполнения вторичного трансфера игрок должен успешно завершить три квеста. Первый из трех квестов может быть начат только персонажем 35 уровня или выше. Если вы хотите увидеть более подробную информацию, щелкните интересующий вас класс.
