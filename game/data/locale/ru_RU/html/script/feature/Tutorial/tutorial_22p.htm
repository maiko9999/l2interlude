[Переход на класс Темный Волшебник]:
Темный Волшебник, достигший 40 уровня, может приступить ко второму переходу класса, чтобы стать Певцом Заклинаний или Призрачным Призывателем. Для выполнения второго перехода класса игрок должен успешно завершить три квеста. Первый из квестов может быть начат только персонажем 35 уровня и выше. Если вы хотите увидеть более подробную информацию, пожалуйста, кликните по классу, который вас интересует.
