[Разговор с помощником для новичков]

Теперь попробуйте поговорить с помощником для новичков.
Поместите указатель мыши на помощника для новичков и нажмите левой кнопкой мыши. Затем указатель мыши изменится на форму воздушного шара. В это время, если вы снова щелкнете левой кнопкой мыши, начнется разговор.

Если вы двигаетесь в направлении стрелки над вашим персонажем, вы встретите помощника для новичков. Помощники будут давать вам полезные инструкции, а также полезные подарки, так что обязательно пообщайтесь с помощником для новичков.
