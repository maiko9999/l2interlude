[Перевод Класса Эльфийского Рыцаря]:
Эльфийский Рыцарь, достигший уровня 40, может приступить к вторичному классовому переходу, чтобы стать Храмовым Рыцарем или Певцом Мечей. Для выполнения вторичного классового перехода игрок должен успешно завершить три задания. Первое из трех заданий может быть выполнено только персонажем уровня 35 и выше. Если вы хотите увидеть более подробную информацию, пожалуйста, выберите интересующий вас класс.
