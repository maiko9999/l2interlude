[Переход на класс Шилиен Оракул]:
Шилиен Оракул, достигший 40 уровня, может приступить ко второму переходу класса, чтобы стать Шилиен Старейшиной. Для того чтобы стать Шилиен Старейшиной, игрок должен успешно завершить три квеста. Испытание Пилигрима может быть начато игроком 35 уровня и выше. Этот квест начинается с встречи с Отшельником Сантьяго в фермерском доме около Оркских Казарм в Глудио. Свидетельство Судьбы может быть начато игроком 37 уровня и выше. Этот квест начинается с встречи с Магистром Кайрой из Гиран Касл Тауна. Наконец, Испытание Реформатора может быть начато игроком 39 уровня и выше. Этот квест начинается с встречи с Жрицей Пупиной из Гиран Касл Тауна.
