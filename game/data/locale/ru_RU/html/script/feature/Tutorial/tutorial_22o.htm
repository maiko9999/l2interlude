[Переход на класс Ассасин]:
Ассасин, достигший 40 уровня, может приступить ко второму переходу класса, чтобы стать Отступником Бездны или Призрачным Стрелком. Для выполнения второго перехода класса персонаж должен успешно завершить три квеста. Первый из квестов может быть начат только персонажем 35 уровня и выше. Если вы хотите увидеть более подробную информацию, пожалуйста, кликните по классу, который вас интересует.
