[Перевод Класса Эльфийского Волшебника]:
Эльфийский Волшебник, достигший уровня 40, может приступить к вторичному классовому переходу, чтобы стать Певцом Заклинаний или Спокровителем Стихий. Для выполнения вторичного классового перехода персонаж должен успешно завершить три задания. Первое из заданий может быть выполнено только персонажем уровня 35 и выше. Если вы хотите увидеть более подробную информацию, пожалуйста, выберите интересующий вас класс.
