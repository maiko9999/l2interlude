[Информация о питомце]:
После достижения 35 уровня персонаж может взяться за квест на получение птенца. Как и его хозяин, птенец будет развиваться с опытом. Когда он достигнет определенного уровня, его можно превратить в страйдера или виверну, сделав его подходящим средством передвижения.
Есть три типа птенцов: птенец ветра, птенец звезд и птенец сумерек. У каждого есть уникальные характеристики. Квест начинается с встречи с Менеджером питомцев Купером из Гиран Касл Тауна.
