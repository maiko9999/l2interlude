Для того чтобы стать Военачальником, игрок должен успешно завершить три квеста. Испытание Вызывающего может быть пройдено игроком 35 уровня и выше. Этот квест начинается с встречи с Верховным Жрецом Холлинтом из Орена Касл Тауна. Наконец, Испытание Чемпиона может быть начато игроком 39 уровня и выше. Этот квест начинается с встречи с Ветераном Аскалоном из Гиран Касл Тауна.
