[Подбор предметов]

Когда вы побеждаете монстров, они сбрасывают свои предметы на землю. Переместите курсор на сброшенный предмет, и курсор изменит свою форму на значок руки, указывающий на то, что предмет можно поднять. Щелкните левой кнопкой мыши по предмету, чтобы поднять его.

Есть еще один способ поднять предмет. Нажмите клавишу F4, и вы автоматически поднимете ближайшие предметы. В данный момент четвертый слот на панели быстрых действий зарегистрирован для действия подбора, и, следовательно, при нажатии клавиши F4 это действие активируется.
