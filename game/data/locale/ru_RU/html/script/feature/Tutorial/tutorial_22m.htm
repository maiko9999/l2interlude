[Перевод Класса Ремесленник]:
Ремесленник, достигший 40-го уровня, может приступить ко второму классовому трансферу, чтобы стать Военным Кузнецом (Кузнец). Для этого игрок должен успешно завершить три задания. Испытание Гильдейца доступно игрокам с уровнем 35 и выше. Начало этого задания связано с встречей с Хранителем Склада Валконом в Гиран Касл Тауне. Свидетельство Процветания может быть получено игроком с уровнем 37 и выше. Начало этого задания связано с встречей с Хранителем Склада Парманом в Гиран Касл Тауне. Наконец, Испытание Маэстро доступно игроку с уровнем 39 и выше. Начало этого задания связано с встречей с Локирином из Деревни Гномов у Железных Врат.
