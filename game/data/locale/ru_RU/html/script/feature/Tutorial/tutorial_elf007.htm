[Разговор с Помощником для Новичков]

Теперь попробуйте поговорить с Помощником для Новичков.
Наведите курсор мыши на Помощника для Новичков и нажмите левой кнопкой мыши. Затем ваш курсор мыши изменится на форму воздушного шара. В этот момент, если вы еще раз нажмете левую кнопку мыши, начнется разговор.

Если вы двигаетесь, следуя указанию стрелки над вашим персонажем, вы наткнетесь на Помощника для Новичков. Помощники будут давать вам полезные инструкции, а также полезные подарки, так что обязательно встречайтесь с Помощником для Новичков.
