[Перевод Класса Эльфийского Оракула]:
Эльфийский Оракул, достигший уровня 40, может приступить к вторичному классовому переходу, чтобы стать Старейшиной. Для того чтобы стать Старейшиной, персонаж должен успешно завершить три задания. Испытание Пилигрима доступно игрокам с уровнем 35 и выше. Начало этого задания связано с встречей с Отшельником Сантьяго в доме около Оркского лагеря в Глудио. Свидетельство Жизни может быть получено игроком с уровнем 37 и выше. Начало этого задания связано с встречей с Мастером Кардиеном в Дион Касл Тауне. Наконец, Испытание Исцелителя доступно игроку с уровнем 39 и выше. Начало этого задания связано с встречей с Священником Банделлосом в Гиран Касл Тауне.
