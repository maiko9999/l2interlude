[Перевод Класса Эльфийского Разведчика]:
Эльфийский Разведчик, достигший уровня 40, может приступить к вторичному классовому переходу, чтобы стать Скитальцем по равнинам или Серебряным Рейнджером. Для выполнения вторичного классового перехода персонаж должен успешно завершить три задания. Первое из трех заданий может быть выполнено только персонажем уровня 35 и выше. Если вы хотите увидеть более подробную информацию, пожалуйста, выберите интересующий вас класс.
