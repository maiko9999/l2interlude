[Переход на класс Рыцарь Палуса]:
Рыцарь Палуса, достигший 40 уровня, может приступить ко второму переходу класса, чтобы стать Рыцарем Шилиена или Танцором Клинка. Для выполнения второго перехода класса игрок должен завершить три квеста. Первый квест может быть выполнен только персонажем 35 уровня и выше. Если вы хотите увидеть более подробную информацию, пожалуйста, кликните по классу, который вас интересует.
