[Перевод Класса Эльфийского Разведчика]:
Для того чтобы стать Серебряным Рейнджером, Эльфийский Разведчик должен успешно завершить три задания. Испытание Искателя доступно только игрокам с уровнем 35 и выше. Начало этого задания связано с встречей с Мастером Дуфнером из Гиран Касл Тауна. Свидетельство Жизни может быть получено только игроком с уровнем 37 и выше. Начало этого задания связано с встречей с Мастером Кардиеном из Диона Касл Тауна. Наконец, Испытание Стрельца доступно только игроку с уровнем 39 и выше. Начало этого задания связано с встречей с Президентом Гильдии Бернардом в Деревне Охотников.
