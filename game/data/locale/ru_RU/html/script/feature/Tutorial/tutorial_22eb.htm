[Перевод Класса Эльфийского Рыцаря]:
Для того чтобы стать Певцом Мечей, Эльфийский Рыцарь должен успешно завершить три задания. Испытание Соперника доступно игроку с уровнем 35 и выше. Начало этого задания связано с встречей с Кашем из Диона Касл Тауна. Свидетельство Жизни может быть получено только игроком с уровнем 37 и выше. Начало этого задания связано с встречей с Мастером Кардиеном из Диона Касл Тауна. Наконец, Испытание Дуэлянта доступно игроку с уровнем 39 и выше. Начало этого задания связано с встречей с Дуэлянтом Кайеном из Орена Касл Тауна.
