[Перевод Класса Орка Шамана]:
Орк Шаман, достигший уровня 40, может приступить к вторичному классовому переходу, чтобы стать Повелителем или Крикуном. Для того чтобы завершить вторичный классовый переход, персонаж должен успешно выполнить три задания. Первое из них может быть взято на себя игроком с уровнем 35 или выше. Если вы хотите увидеть более подробную информацию, пожалуйста, кликните по интересующему вас классу.
