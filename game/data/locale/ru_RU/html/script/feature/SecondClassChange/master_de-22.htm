Каждый знак требует свой собственный тест.
Знак Учёного зарезервирован для тех, кто проходит Испытание Учёного. Для квалификации ваш уровень должен быть выше 35. Чтобы пройти это испытание, вам необходимо посетить Магистра Мириэн в храме Города Замка Диона.
Знак Судьбы зарезервирован для тех, кто проходит Свидетельство Судьбы. Для квалификации ваш уровень должен быть выше 37. Чтобы пройти это испытание, отправляйтесь в Гиранскую Крепость и встретьтесь с Темным Эльфом Магистром Кайрой.
И, наконец, Знак Мага предоставляется только тем, кто проходит Испытание Мага. Для квалификации ваш уровень должен быть выше 39. Чтобы пройти это испытание, посетите Барда Рукала в храме Города Замка Диона.
