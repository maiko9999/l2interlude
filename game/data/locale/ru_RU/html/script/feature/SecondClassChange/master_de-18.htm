Каждый знак требует свой собственный тест.
Знак Искателя зарезервирован для тех, кто проходит Испытание Искателя. Для квалификации ваш уровень должен быть выше 35. Чтобы пройти это испытание, вам необходимо посетить Мастера Дуфнера в Гиранском Замке.
Знак Судьбы зарезервирован для тех, кто проходит Свидетельство Судьбы. Для квалификации ваш уровень должен быть выше 37. Чтобы пройти это испытание, отправляйтесь в Гиранскую Крепость и встретьтесь с Темным Эльфом Магистром Кайрой.
И, наконец, Знак Стрельца предоставляется только тем, кто проходит Испытание Стрельца. Для квалификации ваш уровень должен быть выше 39. Чтобы пройти это испытание, посетите Президента Гильдии Бернарда в Деревне Охотников.
