Руководство для новичков:
Префект Брукурсе, один из твоих Мастеров, стремится поймать тех коварных мараку оборотней, которые убили вестника Владыки Пламени Какая. Мараку оборотни довольно сильны, поэтому не пытайся выполнить это задание, пока не достигнешь 10 уровня. Ты можешь найти мараку оборотней, покидая город по южной дороге и направляясь на юго-восток.
Если ты хочешь больше заданий, отправляйся и ищи их сам. Обязательно поговори с Вождями в Доме Короля.
