Руководство для новичков:
Одноручные мечи — самые распространенные оружия. Их возможности хорошо сбалансированы во всех отношениях. Меч Стража, с которым ты начал свои приключения, относится к этому типу. Если ты не уверен, какое оружие подойдет тебе лучше всего, одноручный меч всегда подходящий выбор.
Ты можешь использовать навык Мощный удар с одноручными мечами. Как ты знаешь, это значительно увеличит твой урон. Основная тактика для тех, кто использует одноручный меч, заключается в том, чтобы начать с Мощного удара, а затем сразу же совершить удар.
