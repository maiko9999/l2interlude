Руководство для новичков:
Вы уже задумывались о следующем оружии, которое вы хотите приобрести? Бродские мечи для вас сейчас просто детские игрушки. На 12-м или 13-м уровне Гладиус - отличный выбор для меча, Кинжал - для кинжала или Гномий Булава - для тупого оружия. Вы можете найти эти оружия в магазинах, но они очень дорогие. Вы должны смочь получить гораздо более выгодную цену, сотрудничая с другими искателями приключений.
