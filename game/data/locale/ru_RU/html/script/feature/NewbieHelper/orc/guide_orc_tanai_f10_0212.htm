Руководство для новичков:
Одноручные тупые оружия имеют более высокую точность, чем другие виды оружия. Однако вероятность нанести критический урон у них намного ниже. Тупые оружия разработаны для надежного нанесения предсказуемого урона. Я рекомендую эти виды оружия Мистикам, которые не обладают обучением для высокой точности в бою, или любому воину, ценящему последовательность в битве.
По мере продвижения, ты сможешь приобрести навыки, специфически предназначенные для тупых оружий. Еще одна важная вещь, которую следует иметь в виду о таких оружиях, это их больший диапазон урона по сравнению с другими типами оружия. Например, тупое оружие с средней силой атаки 10 будет иметь диапазон урона от 7 до 13, в то время как меч с той же средней силой будет иметь диапазон от 9 до 11.
