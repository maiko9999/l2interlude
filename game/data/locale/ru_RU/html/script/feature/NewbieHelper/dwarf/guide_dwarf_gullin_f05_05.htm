Руководство для новичков:
Несколько жителей деревни могут понадобиться тебе в помощь. Поговори с ними и нажми на Задание, чтобы узнать, какие миссии они предлагают.
Конечно, даже если они ищут помощи, они не предложат тебе миссию, которая превышает твои возможности. Если они отказываются предложить тебе что-то, это означает, что тебе нужно набрать еще опыта. Никому не хочется, чтобы на их совести была смерть новичка. Ха!
Ах да, и помни - некоторые квесты можно выполнить только один раз, но другие можно повторять несколько раз.
Я хочу узнать больше о квестах.
