Руководство для новичков:
Нескольким жителям деревни нужна твоя помощь. Поговори с ними и нажми на Квест, чтобы узнать, какие задания они могут предложить.
Конечно, даже если они ищут помощь, они не предложат тебе задание, которое тебе не по силам. Если они отказываются что-то предложить, это значит, что тебе нужно сначала набраться опыта. Никто не хочет, чтобы ты попал в неприятности, из которых не сможешь выбраться.
И помни — некоторые квесты можно выполнить только один раз, а другие можно выполнять несколько раз.
Я хочу узнать больше о квестах.
