Руководство для новичков:
Эй, посмотри на тебя! Ты сделал отличные успехи, мой друг! Теперь я собираюсь научить тебя некоторым новым вещам, поскольку ты готов тренироваться в Шахтах Позвоночника. Сейчас тебе предстоит научиться сотрудничать с другими искателями приключений.
Это поворотный момент в твоей жизни, потому что на 10-м уровне ты сможешь освоить несколько базовых навыков гномов: Создание Предметов, позволяющий создавать предметы по рецептам, а также Поиск и Разложение, которые позволяют собирать сырье с монстров. Тебе нужно научиться максимально эффективно использовать эти навыки.
Овладение этими навыками - самое важное, что тебе нужно сделать прямо сейчас. Ты можешь попытаться разобраться во всем сам, если хочешь, или я могу дать тебе несколько советов, а?
Что мне делать сейчас?
Какое снаряжение лучше всего для меня?
Какое место для охоты лучше всего для меня?
Расскажи мне о навыках.
Какие квесты доступны для меня?
