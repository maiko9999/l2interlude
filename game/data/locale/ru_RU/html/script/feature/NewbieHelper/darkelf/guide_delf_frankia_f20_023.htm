Руководство для новичков:
Ваши приключения уже знакомы с термином душевой стрелой? Вы уже слышали, как я использовал этот термин ранее.
Это форма магического боеприпаса, созданного гномами. Вы можете использовать душевую стрелу как зелья во время вооруженного боя. Они позволяют вам наносить больший урон, но исчезают после использования.
Примерно с 15 уровня вы сможете получить оружие, использующее душевую стрелу. Душевая стрела с рейтингом лучше x1 увеличит урон на указанное количество выстрелов.
