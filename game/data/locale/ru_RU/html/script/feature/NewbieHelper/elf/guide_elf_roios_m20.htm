Руководство для новичков:
Вы сделали замечательные успехи, и мне мало осталось вам чему-то учить. Пришло время для вас покинуть путь новичка и начать свои путешествия в мир за пределами нашего леса. Приготовьтесь отправиться в человеческую деревню Глудин, чтобы совершенствовать свою профессию. Будущее расы эльфов может зависеть от вашей силы. Пусть вам удастся обрести силу, чтобы противостоять перед вами стоящим испытаниям.
Что мне делать теперь?
Какое для меня лучшее снаряжение?
Какое для меня лучшее место для охоты?
Расскажите мне о навыках.
Какие квесты доступны мне?
