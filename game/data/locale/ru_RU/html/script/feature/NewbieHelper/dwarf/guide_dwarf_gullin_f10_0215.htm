Руководство для новичков:
Очевидное преимущество луков - их способность наносить урон на расстоянии, но в них есть и другие характеристики. Луки - самое медленное оружие, но они могут нанести наибольший урон.
Большинство воинов используют свои луки в качестве запасного оружия, хотя эльфы особенно хороши в стрельбе из лука, и многие используют лук в качестве своего основного оружия.
