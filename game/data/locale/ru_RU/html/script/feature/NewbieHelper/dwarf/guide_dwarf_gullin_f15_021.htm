Руководство для новичков:
Думал ли ты о следующем оружии, которое ты хочешь приобрести? Для тебя широкие мечи сейчас просто металлолом. На 12-м или 13-м уровнях Гладиус - отличный выбор для меча, кинжал для кинжала и Гномья булава для дубинки. Ты можешь найти эти оружия в магазинах, но они будут дорогими. Ты должен суметь получить намного лучшую цену, сделав сделку с другими искателями приключений.
Я также должен отметить Короткое Копье, древковое оружие, которое отлично подходит для этих уровней. Древковое оружие очень медленно используется, но оно позволяет атаковать сразу нескольких врагов. Однако, если ты переходишь от своего текущего типа оружия к древковому, ожидай некоторого снижения боевой эффективности, пока ты не освоишь новый тип оружия.
