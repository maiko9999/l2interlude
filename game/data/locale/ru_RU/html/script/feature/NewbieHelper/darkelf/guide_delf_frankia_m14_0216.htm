Руководство для новичков:
Очевидным преимуществом луков является их способность наносить урон на дальней дистанции. Однако есть еще несколько вещей, которые вам следует знать. Луки являются самым медленным видом оружия, но они способны наносить наибольший урон.
Большинство использует свои луки как резервное оружие, но при правильной тренировке лук может стать вашим основным средством атаки.
