Руководство для новичков:
Я расскажу тебе все, что знаю, но не знаю я и всего. Тебе нужно поговорить со всеми жителями, с кем только сможешь, и выслушать, что они имеют тебе сказать.
Ты можешь начать с посещения Мэриз Редбоннет, которая живет недалеко от северных ворот деревни. У этой девушки что-то очень расстроило, так что, возможно, ты сможешь ей помочь.
Также, рядом с южными воротами, Старейшина Спирон из Гильдии Золотого Колеса пытается разобраться с некоторыми проблемами на новой торговой дороге.
