Руководство для новичков:
Слушайте внимательно — вас ждут два важных задания, которые вам следует скоро выполнить. Найдите Тетрарха Тифиэла, когда достигнете 10-го уровня. У него есть важное задание для вас.
Также вам нужно найти Стража Кейлин на южных территориях. Она набирает волонтёров для квеста по подавлению Орков, который вам нужно выполнить, прежде чем вы сможете охотиться на Орков.
