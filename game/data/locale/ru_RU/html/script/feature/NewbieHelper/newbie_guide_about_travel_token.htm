Новичок Помощник:
Новичок Путеводитель предназначен для игроков уровня 20 и ниже.
Он позволяет игроку перемещаться в стартовый город другой расы через Новичков-Путеводителей в каждом городе.
Будьте осторожны, потому что во многих случаях вы можете изучать и выполнять определенные навыки и квесты, которые уникальны для города каждой расы.
Как только игрок достигает уровня 20, он больше не может телепортироваться с помощью Новичкового Путеводного Жетона.
