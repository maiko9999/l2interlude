Руководство для новичков:
Одноручное тупое оружие - традиционное оружие для нас, гномов. Их точность выше, чем у других типов оружия, но вероятность нанесения критического урона намного ниже. Тупые оружия надежно наносят стабильный урон.
По мере вашего продвижения вы сможете изучить навыки, которые работают только с тупыми оружиями. Еще что-то интересное о тупых оружиях - они имеют больший диапазон урона по сравнению с другими типами. Допустим, у вас есть два разных типа оружия с средней атакующей силой 10. Тупое оружие может нанести от 7 до 13 единиц урона за атаку, в то время как меч может нанести от 9 до 11.
