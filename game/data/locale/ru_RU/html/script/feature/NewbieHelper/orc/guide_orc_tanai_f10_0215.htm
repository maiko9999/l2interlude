Руководство для новичков:
Оружие в рукопашном бою, в виде укрепленных перчаток для ударов, требует использования обеих рук. Они выделяются своей скоростью, которая намного лучше, чем у удерживаемого оружия. В других отношениях, таких как точность и шанс нанести критический урон — оба низкие — они напоминают тупые оружия.
Поскольку это двуручное оружие, его нельзя использовать с щитом. Но сила навыка Железные кулаки более чем компенсирует этот недостаток. В любом случае, настоящим бойцам не нужно прятаться за щитом — им не страшно прикасаться к врагу!
