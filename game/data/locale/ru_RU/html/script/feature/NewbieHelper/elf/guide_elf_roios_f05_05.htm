Руководство для новичков:
Будет правильно и похвально оставаться бдительным и искать новые задания. Ведь ваша конечная цель - сражаться за благо всего эльфийского народа? Я расскажу вам о всех квестах, которые я знаю, подходящих для бойцов вашего уровня. Однако могут быть и другие, о которых я не слышал; обращайтесь также к другим жителям деревни, чтобы узнать их потребности.
Я могу сказать вам, что есть квесты, которые можно выполнить только один раз, но также есть повторяющиеся квесты, которые можно выполнять несколько раз.
Хочу узнать больше о квестах.
