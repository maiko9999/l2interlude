Руководство для новичков:
Конечно, тупые оружия всегда предпочтительны для нас, гномов, но если вы предпочитаете что-то с лезвием, длинный меч или кинжал будут хорошим выбором для кого-то вашего уровня. Даже если вам не особо везло на охоте, вы должны сможете быстро заработать достаточно для купли кинжала.
Если у вас есть лук, Лесной Лук может быть слишком громоздким, если он используется только для дополнения рукопашных атак. Охотничий Лук - лучший выбор для случайного использования.
Что касается доспехов, Костяные нагрудники и Костяные набедренники, вместе с баклером, перчатками, Кожаными ботинками и Деревянным шлемом, составляют эффективный ансамбль, когда вы на 17 уровне.
