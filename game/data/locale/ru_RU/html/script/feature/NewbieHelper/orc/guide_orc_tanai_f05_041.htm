Руководство для новичков:
Если ты будешь думать головой, охотясь, то к 5 уровню у тебя должно быть около 60 SP. Префекты могут рассказать тебе о навыках, которые ты можешь изучить.
Навыки делятся на активные и пассивные. Активные навыки требуют их намеренного использования, в то время как пассивные навыки увеличивают твои базовые способности с момента их изучения. На 5 уровне тебе захочется сначала выучить активные атакующие навыки. Мощный удар и Железный кулак — твои варианты. Мощный удар для тех, кто сражается мечом, Железный кулак для рукопашных бойцов.
