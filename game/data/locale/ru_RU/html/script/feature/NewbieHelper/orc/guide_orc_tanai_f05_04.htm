Руководство для новичков:
Ты можешь начать изучать навыки, когда достигнешь 5 уровня. Ты получишь обучение новым навыкам в Гильдии Воинов, которая находится в комнате справа, когда войдешь в Дом Короля, самое большое здание в деревне, прямо в центре города. Префекты там обучат тебя твоим навыкам.
Теперь обрати внимание; тебе понадобятся SP для получения навыков. SP означает "очки навыков", и они зарабатываются вместе с опытом всякий раз, когда ты побеждаешь монстра. SP вычитаются каждый раз, когда ты изучаешь новый навык. Ты можешь проверить свои SP, нажав на кнопку Состояние персонажа, затем на вкладку Статус в окне. Конечно, количество SP, которое тебе потребуется, будет варьироваться в зависимости от навыка, который ты хочешь изучить.
