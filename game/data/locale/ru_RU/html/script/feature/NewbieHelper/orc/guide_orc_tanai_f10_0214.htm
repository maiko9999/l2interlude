Руководство для новичков:
Оружие типа кинжалов позволяет атаковать с непревзойденной скоростью. Более того, у них есть наибольший шанс нанести критический урон. Однако их точность низкая, и, конечно, их дальность довольно короткая. У кинжалов также очень низкий диапазон случайного урона.
Но к сожалению, это оружие имеет мало преимуществ для Орков, которые не могут использовать Смертельный Удар, спасительную характеристику кинжалов.
