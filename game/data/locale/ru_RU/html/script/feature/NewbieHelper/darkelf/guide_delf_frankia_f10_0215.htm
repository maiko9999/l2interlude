Руководство для новичков:
Очевидным преимуществом луков является их способность наносить урон на дальнем расстоянии. Однако есть еще несколько моментов, которые вам следует знать. Луки являются самыми медленными оружиями, но они способны наносить самый большой урон.
Популярная тактика заключается в том, чтобы начать бой с Мощного Выстрела, прежде чем приблизиться для атаки основным оружием.
Большинство бойцов используют свои луки как запасное оружие, но с правильной тренировкой лук может стать вашим основным средством атаки.
