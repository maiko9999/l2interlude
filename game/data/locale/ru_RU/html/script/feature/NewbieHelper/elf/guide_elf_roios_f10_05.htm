Руководство для новичков:
Многие из жителей деревни нуждаются в вашей помощи, и квесты, которые они предлагают, могут принести вам пользу. Не забывайте искать их.

Торговец Хербиел, в магазине на западной стороне города, собирает мешочки спор для исследовательских целей. Обязательно пообщайтесь с ним.
Часовой Кенделл у юго-восточных ворот снова набирает воинов для битвы против племён орков, пришедших в наши земли из-за Туманных Гор. В этот раз план заключается в подавлении главных командиров орков.
