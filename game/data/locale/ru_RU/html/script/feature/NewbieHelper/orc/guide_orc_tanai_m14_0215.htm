Руководство для новичков:
Рукопашное оружие, в виде укрепленных перчаток для ударов, требует обеих рук для использования. Они выделяются своей скоростью, которая гораздо лучше, чем у других видов оружия. В других отношениях, таких как высокая точность и низкая вероятность критического урона, они напоминают тупое оружие.
