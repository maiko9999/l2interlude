Руководство для новичков:
Одноручные мечи являются самыми распространенными оружиями. Их возможности хорошо сбалансированы во всех отношениях. Меч Стража, с которым вы начали свои приключения, был одним из них. Если вы не уверены, какое оружие будет лучшим для вас, одноручный меч всегда подходящий выбор.
Вы можете использовать навык Мощный Удар с одноручными мечами. Как вы знаете, это существенно увеличит ваш урон. Основная тактика для тех, кто использует одноручный меч, заключается в том, чтобы начать с Мощного Удара и сразу после этого совершить разрез.
